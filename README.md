# Sticky Popout Notes

Safe-obfuscated distribution build for Sticky Popout Notes.

## Manual install

1. Download the release assets.
2. Create this folder in your vault:

   .obsidian/plugins/sticky-popout-notes/

3. Put these files into that folder:

   - manifest.json
   - main.js
   - styles.css
   - versions.json

4. Enable the plugin in Obsidian settings.
